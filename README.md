# NETAME

NETAME (NEw Take on AniME) is a modern anime discovery and assistant web application.

## Features

- Trending and popular anime grids powered by AniList GraphQL.
- AI chat assistant that recommends anime, shows posters, and shares wise quotes.
- Express.js backend with REST wrappers.
- Responsive crystal-glass UI with typewriter effects.

## Local development

```bash
npm install
node server.js
```
Open http://localhost:3000 in your browser.

## Deployment

The app can be deployed to Vercel (static + serverless) or any Node host.
